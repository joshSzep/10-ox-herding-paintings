# Ten Ox-Herding Pictures Slidev Deck

This repository contains a Slidev deck for a dharma talk on the ten ox-herding
paintings.

## Use

Install dependencies:

```sh
npm install
```

Run the deck locally:

```sh
npm run dev
```

Build a static version:

```sh
npm run build
```
